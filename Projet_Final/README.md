# CampusBot

CampusBot est un chatbot universitaire intelligent développé dans le cadre du module Applications Réparties.

## Réalisé par

- Mouad Fassi Fihri
- Benlhabib Mohamed El Ghali

## Objectif

CampusBot permet aux étudiants d'obtenir rapidement des informations concernant :

- les modules ;
- les professeurs ;
- les emplois du temps ;
- les salles ;
- les examens ;
- les annonces ;
- les informations générales de l'université.

## Architecture

Le projet est composé de trois parties principales :

1. Frontend Web
2. Chatbot Service
3. Data Service

## Technologies utilisées

- HTML
- CSS
- JavaScript
- Python
- FastAPI
- JSON
- API REST
