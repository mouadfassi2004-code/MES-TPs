CampusBot

Assistant universitaire intelligent pour les étudiants de l’UEMF

CampusBot aide les étudiants de l’Université Euromed de Fès à accéder rapidement aux informations importantes : modules, professeurs, planning, salles, examens et annonces. L’objectif est de faciliter la vie des étudiants, surtout pendant la période des examens.

Architecture

Le projet utilise une architecture répartie avec un Frontend Web, un Chatbot Service, un Data Service et OpenClaw IA.

Base de données

SQLite est utilisée pour stocker les étudiants, modules, professeurs, planning, examens, salles et annonces.

OpenClaw IA

OpenClaw est utilisé pour répondre aux questions générales ou complexes concernant les études, la programmation et les concepts informatiques.

Technologies utilisées

HTML, CSS, JavaScript, Python, FastAPI, SQLite, API REST et OpenClaw IA.

Fonctionnalités principales

Connexion étudiant, chatbot, consultation des modules, professeurs, planning, examens, annonces et réponses intelligentes.
